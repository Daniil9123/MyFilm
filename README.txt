Это проект в котором можно посмотреть 
Для запуска данного сайта необходимо сделать три шага:
1. Открыть терминал;
2. Ввести туда: npm i
Это необходимо для загрузки package, node и чтобы обновить данные в случаи каких либо изменений в коде
3. Ввести туда: npm run dev
Это сообственно говоря запускает сайт где ты можешь просмотреть фильмы и добавить фильмы.
Вы можете зайти чтобы просмотреть или стать админом для добавления фильмов.

Если вам нужен swagger, то можете ввести в браузерную строку: http://localhost:3000/api-docs/#/

Вам также необходимы данные библиотеки для работы:

bcryptjs - Библиотека для хеширования и проверки паролей с использованием алгоритма bcrypt.

dotenv - Загружает переменные окружения из файла .env в process.env.

ejs - Шаблонизатор, позволяющий генерировать HTML с использованием встроенного JavaScript.

express - Быстрый и минималистичный веб-фреймворк для Node.js.

express-session - Middleware для управления пользовательскими сессиями в приложениях Express.

mysql2 - Клиент MySQL для Node.js с лучшей производительностью и поддержкой Promise.

sequelize - ORM на основе Promise для работы с MySQL, PostgreSQL, SQLite и другими СУБД.

swagger-ui-express - Middleware для отображения Swagger UI и взаимодействия с документацией API.
